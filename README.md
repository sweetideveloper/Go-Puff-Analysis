## 📊 GoPuff Data Analysis Project

This project focuses on analyzing the **GoPuff dataset** to extract meaningful insights and understand business performance. The goal is to explore patterns in sales, customer behavior, and product trends using data analysis techniques.

### 🔍 Project Overview

In this analysis, I performed data cleaning, transformation, and exploratory data analysis (EDA) to uncover key insights that can help in data-driven decision-making.

### 🛠️ Tools & Technologies

* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* Jupyter Notebook

### 📈 Key Tasks Performed

* Data Cleaning and Handling Missing Values
* Exploratory Data Analysis (EDA)
* Data Visualization (Bar Charts, Pie Charts, Scatter Plots)
* Identifying Trends and Patterns

### 💡 Insights

* Analyzed sales trends and product performance
* Identified customer preferences
* Visualized key metrics for better understanding

### 🎯 Objective

To demonstrate practical data analysis skills by working with real-world datasets and presenting insights in a clear and visual manner.
